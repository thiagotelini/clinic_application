# 📱 Clinic Application

Sistema de agendamento médico desenvolvido em Flutter

O **Clinic Application** é um projeto acadêmico desenvolvido com **Flutter**,
com o objetivo de simular um sistema de agendamento de consultas médicas
em uma clínica.

A aplicação permite gerenciar atendimentos de forma simples e eficiente,
com foco em **experiência do usuário, validação de regras de negócio e
testes automatizados**.

## 🎯 Objetivo do Projeto

-   Desenvolver um aplicativo mobile utilizando Flutter\
-   Aplicar boas práticas de desenvolvimento mobile\
-   Implementar validações de regras de negócio\
-   Garantir qualidade com testes automatizados

## 🧠 Como funciona

### Cadastro de Agendamento

-   Nome do paciente\
-   Médico\
-   Data e horário (30 min)\
-   Tipo de atendimento\
-   Forma de pagamento

### Validações

-   Não permitir conflito de horário\
-   Apenas intervalos de 30 min\
-   Não permitir datas no passado
